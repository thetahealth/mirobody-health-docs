本页带你 5 分钟内完成首次调用。开始前需要一个由 [Mirobody 支持](mailto:developer@thetahealth.ai) 开通的邮箱账号，或使用下方公开 [演示账号](#演示账号)。

<Steps>
  <Step title="拿 token">
    两步邮箱码登录：

    <CodeGroup>
    ```bash 发送验证码
    curl -X POST https://mcp.thetahealth.cn/email/login \
      -H "Content-Type: application/json" \
      -d '{"email":"demo1@mirobody.ai"}'
    ```

    ```bash 校验拿 token
    curl -X POST https://mcp.thetahealth.cn/email/verify \
      -H "Content-Type: application/json" \
      -d '{"email":"demo1@mirobody.ai","code":"777777"}'
    ```
    </CodeGroup>

    响应中的 `data.access_token` 即为 30 天有效的 JWT。
  </Step>

  <Step title="发起第一次对话">
    <CodeGroup>
    ```bash curl
    TOKEN="<上一步拿到的 access_token>"

    curl -N -X POST https://mcp.thetahealth.cn/api/chat \
      -H "Authorization: Bearer $TOKEN" \
      -H "Content-Type: application/json" \
      -d '{"question":"你好","agent":"App","provider":"DeepSeekFlash"}'
    ```

    ```python Python
    import requests, json

    resp = requests.post(
        "https://mcp.thetahealth.cn/api/chat",
        headers={
            "Authorization":  f"Bearer {TOKEN}",
            "Content-Type":   "application/json",
        },
        json={"question": "你好", "agent": "App", "provider": "DeepSeekFlash"},
        stream=True,
    )
    for line in resp.iter_lines(decode_unicode=True):
        if line.startswith("data: "):
            evt = json.loads(line[6:])
            if evt["type"] == "reply":
                print(evt["content"], end="", flush=True)
            elif evt["type"] == "end":
                break
    ```

    ```javascript Node.js
    const resp = await fetch("https://mcp.thetahealth.cn/api/chat", {
      method: "POST",
      headers: {
        "Authorization":  `Bearer ${TOKEN}`,
        "Content-Type":   "application/json",
      },
      body: JSON.stringify({ question: "你好", agent: "App", provider: "DeepSeekFlash" }),
    });
    const reader = resp.body.getReader();
    const decoder = new TextDecoder();
    let buf = "";
    while (true) {
      const { value, done } = await reader.read();
      if (done) break;
      buf += decoder.decode(value, { stream: true });
      let idx;
      while ((idx = buf.indexOf("\n\n")) !== -1) {
        const chunk = buf.slice(0, idx);
        buf = buf.slice(idx + 2);
        if (!chunk.startsWith("data: ")) continue;
        const evt = JSON.parse(chunk.slice(6));
        if (evt.type === "reply") process.stdout.write(evt.content);
        if (evt.type === "end") return;
      }
    }
    ```
    </CodeGroup>

    返回 SSE 流，逐行 `data: {...}` —— 含 `id` / `reply` / `costStatistics` / `end` 等事件，详见 [对话 API](/zh/api-reference/chat/chat)。
  </Step>
</Steps>

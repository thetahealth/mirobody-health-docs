---
title: "快速开始"
description: "5 分钟内创建密钥并发起首次 /v1 调用。"
---

Mirobody 的托管 API **兼容 OpenAI**：将任意 OpenAI SDK 指向 `https://mcp.thetahealth.cn/v1`，传入 `mb_live_*` 密钥，智能体即可基于每位终端用户的真实健康数据作答。

<Steps>
  <Step title="创建 API 密钥">
    登录开发者控制台 [**cdm.thetahealth.cn**](https://cdm.thetahealth.cn/)（使用你自己的邮箱，或下方的[演示账户](#demo-accounts)），然后前往 **API Keys → Create**。复制 `mb_live_*` 密钥——它**只显示一次**。

    ```bash
    export MIROBODY_API_KEY="mb_live_..."
    ```
  </Step>

  <Step title="发起首次调用">
    <CodeGroup>
    ```bash curl
    curl https://mcp.thetahealth.cn/v1/chat/completions \
      -H "Authorization: Bearer $MIROBODY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
            "model": "mirobody-flash",
            "messages": [{"role": "user", "content": "How is my fasting glucose trending?"}],
            "user": "demo_diabetes"
          }'
    ```

    ```python Python (OpenAI SDK)
    from openai import OpenAI

    client = OpenAI(api_key="$MIROBODY_API_KEY", base_url="https://mcp.thetahealth.cn/v1")
    resp = client.chat.completions.create(
        model="mirobody-flash",
        messages=[{"role": "user", "content": "How is my fasting glucose trending?"}],
        user="demo_diabetes",
    )
    print(resp.choices[0].message.content)
    ```

    ```javascript Node.js (OpenAI SDK)
    import OpenAI from "openai";

    const client = new OpenAI({
      apiKey: process.env.MIROBODY_API_KEY,
      baseURL: "https://mcp.thetahealth.cn/v1",
    });
    const resp = await client.chat.completions.create({
      model: "mirobody-flash",
      messages: [{ role: "user", content: "How is my fasting glucose trending?" }],
      user: "demo_diabetes",
    });
    console.log(resp.choices[0].message.content);
    ```
    </CodeGroup>

    `user` 字段将每位终端用户隔离为一个[数据主体](/zh/api-reference/overview#multi-tenancy-the-user-field)。详见完整的 [Chat 参考](/zh/api-reference/chat)。
  </Step>

  <Step title="为数据主体写入一些数据">
    全新的数据主体没有任何数据，因此智能体会如实告知。写入一条记录（或通过 [`POST /v1/files`](/zh/api-reference/files) 上传一份报告），然后再次提问：

    ```bash
    curl https://mcp.thetahealth.cn/v1/data \
      -H "Authorization: Bearer $MIROBODY_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
            "user": "demo_diabetes",
            "records": [{"indicator": "fasting_glucose", "value": 5.4, "unit": "mmol/L", "time": "2026-06-16T07:30:00Z"}]
          }'
    ```

    详见 [Data](/zh/api-reference/data) 和 [Files](/zh/api-reference/files)。
  </Step>
</Steps>

## 演示账户

使用以下任意账户登录**控制台**，即可在无需开通自有账户的情况下创建密钥。验证码始终为 `000000`。每个账户预置了不同的临床画像：

| 控制台登录账号 | 画像 |
| --- | --- |
| `demo_pcp@mirobody.example` | 全科 / 初级保健 |
| `demo_cardio@mirobody.example` | 心血管 |
| `demo_anxiety@mirobody.example` | 焦虑 / 心理健康 |
| `demo_knee@mirobody.example` | 骨科（膝关节） |
| `demo_cardio_history@mirobody.example` | 心血管，纵向历史数据 |
| `demo_diabetes@mirobody.example` | 糖尿病 / 血糖趋势 |

<Warning>
这些账户为评估用途而**公开共享**。**请勿**上传真实的个人健康数据——其他评估者可能随时读取、覆盖或清空其中内容。如需私有合作伙伴账户，请邮件联系 [Mirobody Support](mailto:developer@thetahealth.ai)。
</Warning>

需要针对预发布环境测试？将 base URL 换成 `https://test-mcp.thetahealth.cn/v1`，控制台换成 [test-cdm.thetahealth.cn](https://test-cdm.thetahealth.cn/)。

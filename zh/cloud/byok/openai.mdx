[**OpenAI**](https://platform.openai.com/) 是 Mirobody **海外集群** 的默认 LLM 后端，覆盖 GPT-5 / GPT-4o / o-系列推理 / 嵌入。

<Warning>
**中国集群提示**：OpenAI 在中国大陆无直连。需要您**自有**的合规出境通道；Mirobody 不代为提供网络代理。若坚持在 CN 集群使用，建议改走 [DashScope](/zh/cloud/byok/dashscope) 或 [火山方舟](/zh/cloud/byok/volcengine)。
</Warning>

<Note>
**官方资料**：[OpenAI Platform](https://platform.openai.com/) · [模型列表](https://platform.openai.com/docs/models) · [API Keys](https://platform.openai.com/api-keys) · [Pricing](https://openai.com/api/pricing/)
</Note>

## 1. 注册并拿 Key

<Steps>
  <Step title="OpenAI Platform 注册">
    登录 [platform.openai.com](https://platform.openai.com/)。组织（Organization）默认即可。
  </Step>
  <Step title="充值或绑定信用卡">
    OpenAI 预付费模式，先在 **Billing → Payment methods** 充值；新账号有 $5 试用额度。
  </Step>
  <Step title="创建 Secret Key">
    在 [API Keys](https://platform.openai.com/api-keys) 创建 secret key（`sk-...`），仅显示一次，立即保存。
  </Step>
</Steps>

## 2. 端点信息

| 项目 | 值 |
| --- | --- |
| Base URL | `https://api.openai.com/v1` |
| 认证方式 | `Authorization: Bearer sk-...` |
| 区域 | 全球，按区域路由 |
| 兼容协议 | OpenAI 官方原生 |

## 3. 可调用模型

| Mirobody `provider` | 底层 model id | 说明 |
| --- | --- | --- |
| `general` | `gpt-5` / `gpt-5-mini` | GPT-5 系列，强通用 |
| `expert` | `gpt-5-pro` 或 `o3-pro` | 深推理 |
| —（待 Mirobody 海外集群上线） | `gpt-4o`、`gpt-4o-mini` | 多模态（图像/语音） |
| — | `o3` / `o4-mini` | 推理系列 |
| — | `text-embedding-3-large` / `text-embedding-3-small` | 嵌入 |

详细模型清单见 [OpenAI Models](https://platform.openai.com/docs/models)。

## 4. 在 Mirobody 请求中使用（海外集群）

```bash
curl -N -X POST <global-cluster-base-url>/api/chat \
  -H "Authorization: Bearer $TOKEN" \
  -H "X-LLM-Provider: openai" \
  -H "X-LLM-Api-Key: sk-..." \
  -H "Content-Type: application/json" \
  -d '{"question":"hi","agent":"App","provider":"general"}'
```

## 自有 Base URL（Azure OpenAI / 自托管代理）

如果您使用 Azure OpenAI、自托管 LiteLLM 或其他兼容 OpenAI 协议的网关：

```text
X-LLM-Base-Url: https://your-azure-endpoint.openai.azure.com/openai/deployments/<deployment>
```

Mirobody 会用该 URL 替换 `https://api.openai.com/v1`。

## 故障排查

- `invalid_api_key` → 检查 `sk-...` 是否被吊销，重新生成。
- `insufficient_quota` → OpenAI 账户余额不足或超出月度限额。
- 中国大陆请求超时 → 走不通；改用 DashScope / Volcengine。
- 区域受限 → OpenAI 部分模型对特定地区不可用（如 EU 某些预览版需 opt-in）。

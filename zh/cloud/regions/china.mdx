国内集群是 Mirobody Platform 当前的主要部署，全部数据存储于中国大陆。

## Base URL

```text
https://mcp.thetahealth.cn
```

## 关键能力

- 全部 [API Reference](/zh/api-reference/overview) 列出的 REST 接口与 WebSocket。
- LLM 后端：`DeepSeekFlash` / `DeepSeekPro` / `kimi` / `minimax` / `豆包`（通过 [BYOK](/zh/cloud/byok/index) 自带 key）。

## 前端

- **主前端**：[**mirobody.thetahealth.cn**](https://mirobody.thetahealth.cn/) —— Mirobody 平台的默认 Web 前端，覆盖 Mirobody 的核心能力。
- **分站 CDM**：[**cdm.thetahealth.cn**](https://cdm.thetahealth.cn/) —— 在 Mirobody 基础上增加了**数据记录**能力的扩展版本。合作伙伴可以把任一个作为参考实现 / 演示站，也可以直接二次开发自己的前端。

## 合规

- 数据 100% 存储于中国大陆。
- 符合 PIPL / 数据安全法 / 网络安全法，详见 [合规](/zh/cloud/compliance)。

## 已知限制

- 未启用 `tool-*` / `user-get_events` / `user-get_food_records` / `user-list_medications` / `user-get_medication_details` 工具（Agent 不会调用）。
- `/api/history` 当前不分页，单次返回全部 summary —— 客户端自行截断。

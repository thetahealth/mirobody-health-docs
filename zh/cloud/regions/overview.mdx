Mirobody 提供两个集群，**数据不跨境**：

| 集群 | Base URL | 主前端 | 状态 |
| --- | --- | --- | --- |
| [🇨🇳 中国](/zh/cloud/regions/china) | `https://mcp.thetahealth.cn` | [mirobody.thetahealth.cn](https://mirobody.thetahealth.cn/) | ✅ 已上线 |
| [🌐 海外](/zh/cloud/regions/global) | 暂未对外开放 | [mirobody.thetahealth.ai](https://mirobody.thetahealth.ai/) ✅ | 🚧 API 筹备中 |

集群在签约时即确定，**无法**在运行时切换。详细合规说明见 [Compliance](/zh/cloud/compliance)。

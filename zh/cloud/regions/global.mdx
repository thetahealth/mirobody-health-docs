## 集群信息

- **主前端**：[mirobody.thetahealth.ai](https://mirobody.thetahealth.ai/) — 已上线
- **API base URL**（`mcp.thetahealth.ai`）：**暂未对外提供服务**，上线时同步公布
- **LLM 后端**：OpenAI / Gemini / Anthropic

## 与国内集群的差异

- 数据存储于美国境内，不跨境。
- 合规框架：HIPAA + GDPR（计划，详见 [海外合规](/zh/cloud/compliance)）。
- 默认启用 OpenAI / Gemini tools；不需要 BYOK header 调用 OpenAI / Gemini 时也支持。

具体上线时间请关注 Mirobody 官方公告或发送邮件至 [Mirobody 支持](mailto:developer@thetahealth.ai) 确认。

## 概览

Mirobody Health 可以连接到外部 MCP 服务器，让您的 AI Agent 能够访问成千上万的额外工具和资源。这解锁了诸如文件系统访问、GitHub 集成、数据库查询等更多能力。

<Info>
模型上下文协议 (Model Context Protocol, MCP) 是一个开放标准，使 AI 助手能够安全地连接到外部数据源和工具。欲了解更多信息，请访问 [modelcontextprotocol.io](https://modelcontextprotocol.io)。
</Info>

## 添加 MCP 服务器

通过 Web 界面向 Mirobody Health 添加自定义 MCP 非常简单。

<Steps>
  <Step title="导航到 Drive 页面">
    打开 Mirobody Health Web 界面，进入用于管理 MCP 连接的 Drive 页面。
  </Step>

  <Step title="点击添加 (Add)">
    点击 "Add" 按钮开始添加新的 MCP 服务器。

    <Frame>
      <img src="/images/add-mcp.png" alt="Drive 界面中的添加 MCP 按钮" />
    </Frame>
  </Step>

  <Step title="配置 MCP 详情">
    填写 MCP 服务器配置信息：
    - **Name**: 此 MCP 服务器的友好名称
    - **MCP Server URL**: 您的 MCP 服务器可访问的 HTTP URL（例如：`https://your-domain.com/mcp`）
    - **Token** (可选): 如果您的 MCP 服务器需要基于 Token 的身份验证，请填写 Token

    <Frame>
      <img src="/images/add-mcp-detail.png" alt="包含服务器详情的 MCP 配置表单" />
    </Frame>

  </Step>

  <Step title="保存并连接">
    点击 "Save" 添加 MCP 服务器。Mirobody Health 将自动执行以下操作：
    - 验证配置
    - 启动 MCP 服务器进程
    - 注册可用的工具和资源
    - 使其对您的 AI Agent 可用

    <Check>
    您的 MCP 服务器现在已连接，其工具已可供 AI Agent 使用！
    </Check>
  </Step>
</Steps>

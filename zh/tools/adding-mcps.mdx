## 概览

Model Context Protocol（MCP）是一个开放标准，让 AI 应用得以连接工具和数据源。了解更多请访问 <a href="https://modelcontextprotocol.io">modelcontextprotocol.io</a>。

Mirobody 的角色是**一个 MCP 服务器**：它通过 [`/mcp` 端点](/zh/tools/mcp-integration) 暴露其 [内置工具](/zh/tools/built-in)，而外部客户端 —— Claude Desktop、Cursor 或任何 MCP 客户端 —— 连接*到*它。

<Card title="将客户端连接到 Mirobody" icon="plug" href="/zh/tools/mcp-integration">
  把 Claude、Cursor 或自定义客户端指向 `/mcp` 端点。
</Card>

## 连接外部 MCP 服务器

引擎不包含对外的 MCP 客户端 —— 没有注册*第三方* MCP 服务器、让其工具进入 Agent 工具集的途径。要扩展 Mirobody 自身 Agent 的能力，请改用 C++ 添加一个**内置工具**。

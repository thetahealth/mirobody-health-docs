---
title: "更新 / 删除 / 重处理日记"
description: "日记条目的生命周期操作"
---

## 更新

```http
POST /api/v1/holywell/journal/update
Authorization: Bearer <access_token>
Content-Type: application/json
```

```json
{
  "id":         "<session_id>",
  "text_input": "更新后的文本（可选）",
  "files":      [ /* 可选，新文件列表 */ ],
  "symptom":    [ /* 可选，症状数据 */ ],
  "ts":         1779431900000
}
```

## 删除

```http
POST /api/v1/holywell/journal/delete
```

```json
{ "id": "<session_id>" }
```

或：

```json
{ "journal_id": 109 }
```

## 标已读

```http
POST /api/v1/holywell/journal/mark_read
```

```json
{ "session_id": "..." }
```

## 重处理（重跑 AI）

```http
POST /api/v1/holywell/journal/reprocess
Authorization: Bearer <access_token>
Content-Type:  application/json
```

```json
{ "id": "<session_id>" }
```

重跑分类 / 摘要 / 症状抽取。前一次失败或切换 LLM 时可用。

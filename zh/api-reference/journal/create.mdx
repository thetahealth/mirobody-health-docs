---
title: "创建日记"
description: "POST /api/v1/holywell/journal/create — 需 BYOK（异步 AI 处理）"
---

<Warning>
**此接口需要 LLM 凭证**（异步 AI 用于分类 / 摘要 / 症状抽取）。调用时需携带 `X-LLM-Provider` 与 `X-LLM-Api-Key`，详见 [BYOK 协议](/zh/cloud/byok/index)。
</Warning>

## 端点

```http
POST /api/v1/holywell/journal/create
Authorization:   Bearer <access_token>
X-LLM-Provider:  dashscope | volcengine | openai | gemini
X-LLM-Api-Key:   <您的 LLM Key>
Content-Type:    application/json
```

```json
{
  "id":         "<可选，session_id，用于幂等>",
  "text_input": "今天感觉精力充沛",
  "files":      [ /* ChatFileObject 列表，结构同 /api/chat */ ],
  "ts":         1779431900000
}
```

要求 `text_input` 和 `files` **至少有一个非空**。

## 响应

```json
{
  "code": 0, "msg": "ok",
  "data": {
    "input":        { "content":"今天感觉精力充沛", "file_list":[], "ts":1779431900000, "msg_ids":[] },
    "ts":           1779431900000,
    "session_id":   "today_504_2026_05_22_1779431370215",
    "read_status":  0,
    "write_status": 1,
    "ai_status":    0,
    "journal_id":   109,
    "summary":      null,
    "symptom":      null,
    "version":      1
  }
}
```

后端立即写入日记行，AI 处理异步进行。轮询 [`/journal/list`](/zh/api-reference/journal/list)，看 `ai_status` 变 `1` 即处理完。

**推荐轮询间隔**：2~5 秒，最多 60 秒。

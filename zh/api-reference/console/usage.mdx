---
title: "用量"
description: "GET /api/console/usage —— 请求数、令牌数与费用。"
---

读取你账户的用量。使用你的**控制台登录会话（JWT）**进行鉴权。

```http
GET /api/console/usage?days=30
Authorization: Bearer <console_jwt>
```

| Query | Description |
| --- | --- |
| `days` | 聚合的时间窗口（例如 `30`）。 |

{/* TODO(billing): 全球集群的结算币种尚未定 —— 在产品确认前保持币种中性的表述。 */}
返回该时间窗口内的请求数、令牌总量与计量费用。计费按**实际令牌用量**进行；控制台是最终费用的权威来源。控制台会在 **Usage** 下以图表形式呈现这些数据。

```bash
curl "https://platform.mirobody.ai/api/console/usage?days=30" \
  -H "Authorization: Bearer $CONSOLE_JWT"
```

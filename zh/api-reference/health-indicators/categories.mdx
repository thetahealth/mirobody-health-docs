---
title: "类别与数据源枚举"
description: "GET /api/v1/holywell/health-indicator/categories"
---

返回健康指标系统所用的类别与数据源枚举。

```http
GET /api/v1/holywell/health-indicator/categories
Authorization: Bearer <access_token>
```

```json
{ "code": 0, "msg": "success", "data": {
  "category": ["medication","encounter","lab_results","exam_reports","food","other"],
  "source":   ["chat","journal","excel","document","cda","ehr","device","other"]
} }
```

调用 [`POST /health-indicator/watch`](/zh/api-reference/health-indicators/query) 时，过滤条件用这里的枚举值。

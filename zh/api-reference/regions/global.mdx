**基于 `mirobody.ai` 的全球集群**是 Mirobody 的主环境。

## 主机地址

| 角色 | 地址 |
| --- | --- |
| **API**（`base_url`） | `https://api.mirobody.ai/v1` |
| **开发者控制台** | [platform.mirobody.ai](https://platform.mirobody.ai/) |
| **网页应用** | [chat.mirobody.ai](https://chat.mirobody.ai/) |

## 集群信息

- **LLM 提供方**：由 Mirobody 托管（`mirobody-flash` / `mirobody-expert`）—— 无需你自带 LLM 密钥。
- **区域处理**：全球环境。发送受监管数据前，请先确认合同中的数据驻留要求。
- **隐私控制**：Subject 隔离、显式留存与自助删除 —— 见[隐私与合规](/zh/api-reference/compliance)。

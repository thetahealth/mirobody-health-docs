## 集群信息

- **LLM 提供方**：由 Mirobody 托管（`mirobody-flash` / `mirobody-expert`）—— 无需你自带 LLM 密钥。
- **数据驻留**：美国，无跨境复制。
- **合规**：HIPAA + GDPR（规划中）—— 详见[合规](/zh/api-reference/compliance)。

## 欧盟（EU）——即将推出

我们还规划了**欧盟 / 欧洲专属部署**，面向需要数据留在欧盟境内的合作伙伴。与 Mirobody 的每个集群一样，它为**在地、合规的部署**而设计：

- **欧盟数据驻留**——无跨境传输；符合 **GDPR**。
- **本地化部署**——Mirobody 可在地运行（由 Mirobody 托管或合作伙伴自托管），健康数据不出辖区。
- 与其它集群相同的 OpenAI 兼容 `/v1` 契约。

关于上线时间，或希望登记欧盟 / 本地化部署的意向，请关注 Mirobody 公告，或邮件联系 [Mirobody Support](mailto:developer@thetahealth.ai)。

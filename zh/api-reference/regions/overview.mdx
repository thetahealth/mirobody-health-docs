Mirobody 采用**全球优先**架构，各集群之间**无跨境数据流动**。本文档中的代码示例统一使用全球集群的 `base_url`；替换为你签约集群的地址即可。

| 集群 | API base URL | 控制台 | 网页应用 | 状态 |
| --- | --- | --- | --- | --- |
| [🌐 全球](/zh/api-reference/regions/global) | `https://api.mirobody.ai/v1` | [platform.mirobody.ai](https://platform.mirobody.ai/) | [chat.mirobody.ai](https://chat.mirobody.ai/) | ✅ |
| [🇨🇳 中国](/zh/api-reference/regions/china) | `https://api.mirobody.cn/v1`（即将上线） | [platform.mirobody.cn](https://platform.mirobody.cn/) | [chat.mirobody.cn](https://chat.mirobody.cn/) | 🚧 仅 API 待上线 |

集群在签约时确定；**不支持运行时切换**。详见[合规](/zh/api-reference/compliance)。

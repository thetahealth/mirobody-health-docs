Mirobody 采用**全球优先**架构，各集群之间**无跨境数据流动**。本文档中的每个代码示例都指向已上线的**全球测试**集群；待其它集群上线后，只需替换 `base_url` 即可。

| 集群 | API base URL | 控制台 | C 端应用 | 状态 |
| --- | --- | --- | --- | --- |
| [🌐 全球](/zh/api-reference/regions/global) | 测试 `https://test-mirobody-api.thetahealth.ai/v1` · 生产 `mirobody-api.thetahealth.ai/v1` | [platform.mirobody.ai](https://platform.mirobody.ai/) | [chat.mirobody.ai](https://chat.mirobody.ai/) | ✅ 已上线（测试）· 生产筹备中 |
| [🇨🇳 中国](/zh/api-reference/regions/china) | `https://mirobody-api.thetahealth.cn/v1` | [cdm.thetahealth.cn](https://cdm.thetahealth.cn/) | [mirobody.thetahealth.cn](https://mirobody.thetahealth.cn/) | ✅ 测试应用已上线 · 🚧 API 筹备中 |
| [🇪🇺 欧盟](/zh/api-reference/regions/global) | 尚未对外开放 | — | — | 🚧 即将推出 |

集群在签约时确定；**不支持运行时切换**。详见[合规](/zh/api-reference/compliance)。

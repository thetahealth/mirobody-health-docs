中国集群将 100% 的数据留存于中国大陆境内，无跨境复制。

## 主机地址

| 界面 | 地址 |
| --- | --- |
| **API**（`base_url`） | `https://mirobody-api.thetahealth.cn/v1` |
| **开发者控制台** | [cdm.thetahealth.cn](https://cdm.thetahealth.cn/) |
| **网页应用** | [mirobody.thetahealth.cn](https://mirobody.thetahealth.cn/) |

控制台是你创建 `mb_live_*` 密钥、查看用量并试用 Playground 的地方。Mirobody 网页应用是旗舰产品——与本 API 背后是同一套引擎。

## 模型

由 Mirobody 托管——无需你自带 LLM 密钥：`mirobody-flash`（默认）/ `mirobody-expert`。见[模型](/zh/api-reference/models)。

## 存储与驻留

- 文件存储于**阿里云 OSS（杭州地域）**。
- 100% 数据留存中国大陆境内；无跨境复制。

## 什么是 Mirobody Health？

Mirobody Health 是一个 **开源、AI 原生的数据引擎**：把丰富的健康上下文与最新 AI 能力连接起来，帮助你构建更个性化、更全面的健康 AI 体验。

它可以连接多种设备、处理文件（检验报告、医疗文档等），并通过 MCP/Tools/Skills 支持提供自治 Agent 能力，同时内置 Auth/Frontend/MCP server 等能力——非常适合在保护隐私的前提下管理个人健康数据，或搭建定制化的健康 AI 应用。

## 核心特性

<AccordionGroup>
  <Accordion title="集成外部健康数据" icon="link">
    连接可穿戴设备、硬件设备以及医疗记录（后续支持）。借助我们的集成框架，你可以在数小时内 Bring Your Own Data Provider (BYOP)。支持与 Garmin、Whoop 等的 OAuth 安全连接。
  </Accordion>

  <Accordion title="文件处理与管理" icon="file">
    支持多模态文档：PDF、Excel、照片（PNG、JPEG、WebP、HEIC、HEIF）以及音频（WAV、MP3、AIFF、AAC、OGG、FLAC）。让 Agent 能够访问文件中的上下文，从检验报告到手写笔记。
  </Accordion>

  <Accordion title="原生支持 Tools、MCPs 与 Skills" icon="wrench">
    内置面向健康场景的 Tools。你可以用 Python 添加自定义 Tools，无需复杂 JSON schema 或手工绑定。支持连接第三方 MCPs，并可直接加载标准 Claude Agent Skills（后续支持）。
  </Accordion>

  <Accordion title="用 SOTA Agentic 框架进行对话" icon="comments">
    强大的 agent harness 可在本地复刻 Claude Code 的自治能力：智能调用 Tools，并结合相关健康上下文进行深度研究与洞察。
  </Accordion>

  <Accordion title="MCP Server 与 ChatGPT App" icon="bolt">
    可作为支持 OAuth 的 MCP Server 连接 Claude、Cursor 或自定义客户端；也可作为 GPT app 在 ChatGPT 内启用相关能力。
  </Accordion>

  <Accordion title="随处部署，完全可控" icon="server">
    所有组件均开源，可在本地/自建环境中运行并保护隐私。可通过 OpenRouter 或各家官方 key 接入你自己的 LLM provider。使用 Docker 一行命令即可安装。
  </Accordion>
</AccordionGroup>

## 技术栈

Mirobody Health 使用现代、可用于生产环境的技术构建：

- **Python 3.12+**：使用 async/await 实现高性能
- **FastAPI**：快速、现代的 API Web 框架
- **PostgreSQL**：稳健的关系型数据库，用于数据存储
- **Redis**：高性能缓存与会话管理
- **Docker**：容器化部署，确保一致性
- **OAuth 1.0 & 2.0**：行业标准认证协议

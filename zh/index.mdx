## 什么是 Mirobody Health？

Mirobody Health 是一个 **开源、AI 原生的数据引擎**，它将丰富的健康上下文与最新的 AI 能力联系起来，从而实现深度个性化且全面的健康 AI 体验。

它可以连接各种设备、处理文件（化验单、医疗文件）并提供支持 MCP/Tools/Skills 的自主 Agent 能力，同时还提供开箱即用的身份验证/前端/MCP 服务器功能——是管理具有隐私保护的个人健康数据或构建定制健康 AI 应用的理想选择。

## 核心特性

<AccordionGroup>
  <Accordion title="集成外部健康数据" icon="link">
    连接可穿戴设备、硬件设备和医疗记录（即将推出）。使用我们的集成框架，在数小时内实现“带上你自己的数据提供商 (BYOP)”。支持与 Garmin、Whoop 等的 OAuth 加密连接。
  </Accordion>

  <Accordion title="文件处理与管理" icon="file">
    支持多模态文档：PDF、Excel 文件、照片（PNG、JPEG、WebP、HEIC、HEIF）和音频（WAV、MP3、AIFF、AAC、OGG、FLAC）。使 Agent 能够访问文件中的上下文，从化验单到手写笔记。
  </Accordion>

  <Accordion title="原生支持 Tools、MCPs 和 Skills" icon="wrench">
    开箱即用，包含专注于健康的工具。使用 Python 添加自定义工具，无需复杂的 JSON Schema 或手动绑定。连接到第三方 MCP。直接加载标准 Claude Agent Skills（即将推出）。
  </Accordion>

  <Accordion title="使用 SOTA Agent 框架进行对话" icon="comments">
    强大的 Agent 容器在本地复制了 Claude Code 的自主能力。智能利用工具并利用相关的健康上下文进行深度研究和洞察。
  </Accordion>

  <Accordion title="MCP 服务器和 ChatGPT 应用" icon="bolt">
    作为支持 OAuth 的 MCP 服务器运行，连接到 Claude、Cursor 或自定义客户端。同时也可作为 GPT 应用运行，在 ChatGPT 中启用相关能力。
  </Accordion>

  <Accordion title="随处部署，完全掌控" icon="server">
    所有组件均为开源，可在本地运行并提供完全的隐私保护。通过 OpenRouter 或第一方密钥接入你自己的 LLM 提供商。使用 Docker 一行命令安装。
  </Accordion>
</AccordionGroup>

## 技术栈

Mirobody Health 使用现代且可用于生产环境的技术构建：

- **Python 3.12+**：使用 async/await 模式实现高性能
- **FastAPI**：用于构建 API 的快速、现代 Web 框架
- **PostgreSQL**：用于数据存储的稳健关系型数据库
- **Redis**：高性能缓存和会话管理
- **Docker**：实现一致性的容器化部署
- **OAuth 1.0 & 2.0**：行业标准身份验证协议

## Endpoint

```text
wss://mcp.thetahealth.cn/ws/upload-with-llm-analysis?token=<access_token>
```

## Differences from `/ws/upload-health-report`

- Same upload protocol (`upload_start` / `upload_chunk` / `upload_end`).
- After upload, runs the full LLM analysis pipeline in one go — designed for the "upload and immediately get a complete report" UX.

Server response sequence includes all upload events plus a final, more detailed AI summary embedded in `upload_completed.analysis`.

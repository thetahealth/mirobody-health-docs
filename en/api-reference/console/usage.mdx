Read your account's usage. Authenticates with your **console login session (JWT)**.

```http
GET /api/console/usage?days=30
Authorization: Bearer <console_jwt>
```

| Query | Description |
| --- | --- |
| `days` | Window to aggregate over (e.g. `30`). |

{/* TODO(billing): settlement currency for the global cluster is not finalized — keep this wording currency-neutral until product confirms. */}
Returns request count, token totals, and metered cost over the window. Billing is per **actual token usage**; the console is the source of truth for final cost. The console renders this as charts under **Usage**.

```bash
curl "https://platform.mirobody.ai/api/console/usage?days=30" \
  -H "Authorization: Bearer $CONSOLE_JWT"
```

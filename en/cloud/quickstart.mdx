This page walks you through your first Mirobody Platform API call in 5 minutes. You need either a partner email provisioned via [Mirobody Support](mailto:developer@thetahealth.ai), or one of the public [demo accounts](#demo-accounts) below.

<Steps>
  <Step title="Get a token">
    Two-step email-code login:

    <CodeGroup>
    ```bash Request code
    curl -X POST https://mcp.thetahealth.cn/email/login \
      -H "Content-Type: application/json" \
      -d '{"email":"demo1@mirobody.ai"}'
    ```

    ```bash Verify & exchange
    curl -X POST https://mcp.thetahealth.cn/email/verify \
      -H "Content-Type: application/json" \
      -d '{"email":"demo1@mirobody.ai","code":"777777"}'
    ```
    </CodeGroup>

    The response's `data.access_token` is a 30-day JWT.
  </Step>

  <Step title="Prepare an LLM key (BYOK)">
    For the China cluster we recommend [Aliyun DashScope](/en/cloud/byok/dashscope) (covers DeepSeek / Kimi / Qwen) or [Volcengine](/en/cloud/byok/volcengine) (Doubao).

    Once you have an API key, **every** AI request must carry two headers:

    ```text
    X-LLM-Provider: dashscope
    X-LLM-Api-Key:  <your LLM key>
    ```

    See the [BYOK protocol](/en/cloud/byok/index).
  </Step>

  <Step title="Make your first call">
    <CodeGroup>
    ```bash curl
    TOKEN="<the access_token from step 1>"
    LLM_KEY="<your DashScope key>"

    curl -N -X POST https://mcp.thetahealth.cn/api/chat \
      -H "Authorization: Bearer $TOKEN" \
      -H "X-LLM-Provider: dashscope" \
      -H "X-LLM-Api-Key: $LLM_KEY" \
      -H "Content-Type: application/json" \
      -d '{"question":"Hello","agent":"App","provider":"DeepSeekFlash"}'
    ```

    ```python Python
    import requests, json
    resp = requests.post(
        "https://mcp.thetahealth.cn/api/chat",
        headers={
            "Authorization":  f"Bearer {TOKEN}",
            "X-LLM-Provider": "dashscope",
            "X-LLM-Api-Key":  LLM_KEY,
            "Content-Type":   "application/json",
        },
        json={"question": "Hello", "agent": "App", "provider": "DeepSeekFlash"},
        stream=True,
    )
    for line in resp.iter_lines(decode_unicode=True):
        if line.startswith("data: "):
            evt = json.loads(line[6:])
            if evt["type"] == "reply":
                print(evt["content"], end="", flush=True)
            elif evt["type"] == "end":
                break
    ```

    ```javascript Node.js
    const resp = await fetch("https://mcp.thetahealth.cn/api/chat", {
      method: "POST",
      headers: {
        "Authorization":  `Bearer ${TOKEN}`,
        "X-LLM-Provider": "dashscope",
        "X-LLM-Api-Key":  LLM_KEY,
        "Content-Type":   "application/json",
      },
      body: JSON.stringify({ question: "Hello", agent: "App", provider: "DeepSeekFlash" }),
    });
    const reader = resp.body.getReader();
    const decoder = new TextDecoder();
    let buf = "";
    while (true) {
      const { value, done } = await reader.read();
      if (done) break;
      buf += decoder.decode(value, { stream: true });
      let idx;
      while ((idx = buf.indexOf("\n\n")) !== -1) {
        const chunk = buf.slice(0, idx);
        buf = buf.slice(idx + 2);
        if (!chunk.startsWith("data: ")) continue;
        const evt = JSON.parse(chunk.slice(6));
        if (evt.type === "reply") process.stdout.write(evt.content);
        if (evt.type === "end") return;
      }
    }
    ```
    </CodeGroup>

    You'll get an SSE stream of `data: {...}` lines — `id` / `reply` / `costStatistics` / `end`. See [Chat API](/en/api-reference/chat/chat) for the full event reference.
  </Step>
</Steps>

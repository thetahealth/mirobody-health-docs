## Overview

The Model Context Protocol (MCP) is an open standard that lets AI applications connect to tools and data sources. Learn more at <a href="https://modelcontextprotocol.io">modelcontextprotocol.io</a>.

Mirobody's role is as an **MCP server**: it exposes its [built-in tools](/en/tools/built-in) over the [`/mcp` endpoint](/en/tools/mcp-integration), and external clients — Claude Desktop, Cursor, or any MCP client — connect *to* it.

<Card title="Connect a client to Mirobody" icon="plug" href="/en/tools/mcp-integration">
  Point Claude, Cursor, or a custom client at the `/mcp` endpoint.
</Card>

## Connecting an external MCP server

The engine does not include an outbound MCP client — there is no way to register a *third-party* MCP server and have its tools appear in the agent's toolset. To extend what Mirobody's own agent can do, add a **built-in tool** in C++ instead.

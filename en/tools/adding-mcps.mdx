## Overview

The Model Context Protocol (MCP) is an open standard that lets AI applications connect to tools and data sources. Learn more at <a href="https://modelcontextprotocol.io">modelcontextprotocol.io</a>.

Mirobody's primary role is as an **MCP server**: it exposes its [built-in tools](/en/tools/built-in) over the [`/mcp` endpoint](/en/tools/mcp-integration), and external clients — Claude Desktop, Cursor, ChatGPT, or any MCP client — connect *to* it.

<Card title="Connect a client to Mirobody" icon="plug" href="/en/tools/mcp-integration">
  Point Claude, Cursor, or a custom client at the `/mcp` endpoint.
</Card>

To extend what Mirobody's own agent can do, add a **built-in tool** in C++ — see [Adding Custom Tools](/en/tools/adding-tools).

## Connecting an external MCP server

<Warning>
**Not verified in the current C++ engine (Mirobody).** The screenshots below describe adding a *third-party* MCP server through the web UI so its tools become available to the agent. This outbound-connection capability could **not be confirmed** against the open-source engine at the time of writing — the confirmed MCP surface is Mirobody acting as an MCP *server* (the `/mcp` endpoint) that clients connect to. Treat this section as *(verify)* and check the running build's web UI before relying on it.
</Warning>

If your build's web UI exposes MCP management, the flow is:

<Steps>
  <Step title="Open the MCP management view">
    In the Mirobody web interface, go to the page where MCP connections are managed.
  </Step>

  <Step title="Add an MCP server">
    Start adding a new MCP server.

    <Frame>
      <img src="/images/add-mcp.png" alt="Add MCP button in the web interface" />
    </Frame>
  </Step>

  <Step title="Configure the server">
    Provide the server details:
    - **Name**: a friendly name for this MCP server
    - **MCP Server URL**: the HTTP URL where the MCP server is reachable (e.g. `https://your-domain.com/mcp`)
    - **Token** (optional): an authentication token, if the server requires one

    <Frame>
      <img src="/images/add-mcp-detail.png" alt="MCP configuration form with server details" />
    </Frame>
  </Step>

  <Step title="Save">
    Save the configuration to register the server's tools.
  </Step>
</Steps>
